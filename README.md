### 简介
------------------
vue-tofu-admin 是专门打造的中后台集成方案，是 vue 和 element-ui 结合的最佳实践。

### 功能
----------------

```
- 登录页

- 权限验证
  - 页面权限
  - 指令权限

- 新手引导

- 全局功能
  - 国际化多语言  -------------------------------------  √
  - 多种动态换肤
  - 动态侧边栏(支持多级路由嵌套)  ----------------------  √
  - 动态面包屑 ----------------------------------------  √
  - 快捷导航(标签页)  ---------------------------------  √
  - mock数据
  - Screenfull全屏  -----------------------------------  √
  - 自适应收缩侧边栏

- 编辑器
  - 富文本
  - Markdown

- Excel
  - 导入
  - 导出

- 表格
  - 动态表格
  - 拖拽表格
  - 树形表格
  - 内联编辑

- 错误页面
  - 403
  - 404

- 引导页
- Markdown to Html
- G2 图表  ---------------------------------------------  √
```
